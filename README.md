# JS Event Loop Simulator

A visual simulator for JavaScript's execution model in TypeScript.

## Status

**Milestone 1: Call Stack — DONE**

Working call stack implementation with:
- StackFrame class (function name, arguments, local variables, return address)
- CallStack class with push/pop/peek/isEmpty/size operations
- Stack overflow detection (configurable max depth)
- Stack underflow error handling
- 20 passing tests, 100% coverage

## Language

TypeScript
